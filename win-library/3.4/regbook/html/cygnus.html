cygnus {regbook}R Documentation

cygnus

Description

자료 cygnus는 백조자리 성운 방향에 있는 47개 별들의 밝기와 표면온도를 조사한 것이다.

Usage

cygnus

Format

이 데이터프레임은 다음과 같은 열로 구성되어 있다.

light

빛의 강도. 로그 변환된 값이다.

temp

표면 온도. 로그 변환된 값이다.

References

Rousseuw & Leroy, 1987

Examples

plot(light ~ temp, cygnus)
