KBO1 {regbook}R Documentation

KBO1

Description

자료 KBO1는 2011년 1군 경기에서 1경기 이상 뛴 타자 162명의 선수들을 대상으로 조사한 것이다.

Usage

KBO1

Format

이 자료는 다음과 같은 변수로 이루어져 있다.

Y

연봉(천만원)

X1

안타수

X2

홈런수

X3

타점수

X4

득점수

X5

볼넷수

X6

통산안타수

X7

통산경력

team

구단명

player

선수명
