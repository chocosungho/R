galton {regbook}R Documentation

galton

Description

골턴의 키 자료 galton.

Usage

galton

Format

이 데이터프레임은 다음과 같은 열로 구성되어 있다.

sheight

아들의 키(inch)

fheight

아버지의 키(inch)

References

Weisberg. 1985.

Examples

  with(galton, {
    son <- sheight * 2.54
    father <- fheight * 2.54
    plot(son ~ father)
  })
